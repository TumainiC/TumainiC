### 📌 Featured Projects

#### 🌍 [Tanzania Water Wells — Predictive Maintenance](https://github.com/TumainiC/Tanzania_Water_Wells_Project)
Supervised ML model predicting water point functionality (functional / needs repair / non-functional) across **59,400 water points**. Compared Logistic Regression, XGBoost, Random Forest, and KNN — achieving 71% accuracy. Built to support the Tanzanian Water Ministry's resource allocation decisions.  
`Python` `Scikit-learn` `XGBoost` `Feature Engineering` `Pandas`

#### 🏙 [SDG 11 Urban Clustering Analysis](https://github.com/TumainiC/sgd11_urban_clustering)
Unsupervised ML solution using K-means clustering to categorise countries by urban slum population patterns, addressing UN SDG 11.1.1. Validated with Silhouette Analysis, Davies-Bouldin Index, and bootstrap stability. Includes an interactive Streamlit dashboard for policymakers.  
`Python` `K-means` `Scikit-learn` `Plotly` `Streamlit` `Seaborn`
